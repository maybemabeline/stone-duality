Definition 1

We say that a subset \(S \subseteq P\) is directed if it is both nonempty and for every \(x, y \in S\) there exists \(z \in S\), such that \(x \leq z\) and \(y \leq z\). We say that \(S\) is downwards closed if whenever \(x \in S\) and \(y \leq x\), we also have \(y \in S\).

If \(S\) is both directed and downwards closed, we call \(S\) an ideal of \(P\).

Lemma 2

For all \(x \in P\), the set \(\downarrow x = \left\{ y \in P \vert y \leq x\right\} \) is an ideal of \(P\).

Lemma 3

Suppose the poset \(P\) has binary joins \(\vee \) and \(S \subseteq P\) is downwards closed. Then \(P\) is directed (and hence an ideal) if and only if \(P\) is closed under binary joins.

Definition 4

Whenever a directed subset \(S \subseteq P\) has a join, we call it a directed join and denote it by \(\bigvee ^{\uparrow }S\). If \(P\) has all directed joins, we call \(P\) a directed complete partial order, or dcpo for short.

Lemma 5

To construct arbitrary joins in a dcpo \(P\), it suffices to construct all finite joins. Furthermore, to show meets distribute over arbitrary joins, it suffices to show they distribute overy directed and finite joins.

Definition 6

We define \(\operatorname{Idl}(P)\) to be the set of all ideals of \(P\) and call it the ideal completion of \(P\). It is easy to see that \(\operatorname{Idl}(P)\) forms a poset under subset inclusion and that there is a monotone function \(\eta : P \to \operatorname{Idl}(P)\), given by \(\eta (x) = \downarrow x\).

Proposition 7

The set \(\operatorname{Idl}(P)\) is a dcpo, with directed joins given by unions.
