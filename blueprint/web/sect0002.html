2 Ideal completion

Definition 11
#

We define \(\operatorname{Idl}(P)\) to be the set of all ideals of \(P\) and call it the ideal completion of \(P\). The set \(\operatorname{Idl}(P)\) forms a poset under subset inclusion.

Proposition 12
#

The assignment \(x \mapsto {\downarrow }x\) defines a monotone function \(\eta : P \to \operatorname{Idl}(P)\).

Proposition 13
#

The union of a directed family of ideals is an ideal. Consequently, \(\operatorname{Idl}(P)\) is a dcpo.

Theorem 14
#

The poset \(\operatorname{Idl}(P)\) forms the free dcpo over \(P\) with unit \(\eta \). Given a dcpo \(Q\) and a monotone function \(f : P \to Q\), there exists a unique Scott continuous function \(\overline{f} : \operatorname{Idl}(P) \to Q\), such that \(\overline{f}(\dowset {x}) = f(x)\) for all \(x \in P\).

The function \(\overline{f}\) is defined by the assignment \(I \mapsto \bigvee ^{\uparrow }\left\{ f(x) \vert x \in I \right\} \).

Corollary 15
#

The assignment \(P \mapsto \operatorname{Idl}(P)\) is functorial.

Any additional joins and meets that exist in \(P\) also exist in \(\operatorname{Idl}(P)\) and are preserved by \(\eta \).

Proposition 16
#

Suppose \(P\) has all finite meets. Then so does \(\operatorname{Idl}(P)\), with binary joins given by intersection and top element given by the ideal \(P\). Since \({\downarrow }(x \wedge y) = {\downarrow }x \cap {\downarrow }y\) and \({\downarrow }1 = P\), the function \(\eta \) preserves all finite meets.

Proposition 17
#

Suppose \(P\) has all finite meets. Then meets distribute over directed joins in \(\operatorname{Idl}(P)\).
